# JP Morgan Financial Risk Analysis

## 📖 Project Overview
This project analyzes JP Morgan customer transaction data to understand customer behavior, transaction trends, and financial risks.

The analysis includes:
- Data Cleaning
- Transaction Analysis
- Customer Segmentation
- Risk Detection
- Hypothesis Testing
- Data Visualization

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy

---

## 📊 Key Insights
- Identified high-risk and dormant accounts
- Analyzed credit vs debit transaction trends
- Detected anomalies using z-score and IQR
- Performed hypothesis testing on customer segments

---

## 📈 Visualizations
- Transaction Trend Analysis
- Histogram
- Boxplot
- Heatmap

---

## 📂 Files Included
- Jupyter Notebook (.ipynb)
- Dataset (.csv)
- Summary PDF

---

## 🚀 Conclusion
This project helps understand customer financial behavior and supports proactive financial risk monitoring.
